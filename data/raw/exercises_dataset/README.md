# Exercises Dataset

Esta pasta contém o arquivo bruto usado no estudo `exercises_dataset`.

## Arquivo

- `final_exercise_dataset.csv`

## Uso no repositório

- Script principal: `scripts/exercises_dataset/exercises_dataset_cmdstanr.R`
- Modelos Stan: `scripts/exercises_dataset/*.stan`
- Saídas regeneráveis: `outputs/exercises_dataset/`

## Origem

Base preparada a partir do Kaggle Exercises Dataset usado no projeto analítico. Verifique os termos da fonte original antes de redistribuição externa.

## Observações

O arquivo é versionado para permitir reprodução imediata após o clone. Derivados, tabelas, diagnósticos e CSVs do CmdStan devem permanecer em `outputs/exercises_dataset/` e não devem ser versionados por padrão.
