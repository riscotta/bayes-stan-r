# Consumer Shopping Trends (raw)

Coloque nesta pasta o CSV bruto baixado do Kaggle para o estudo:

- `scripts/consumer_shopping_trends_beta/consumer_shopping_trends_beta_rstan.R`

Arquivo esperado por padrão:

- `Consumer_Shopping_Trends_2026 (6).csv`

Fonte informada:

- `https://www.kaggle.com/datasets/minahilfatima12328/consumer-shopping-trends-analysis`

Observações:

- este CSV **não é versionado** no repositório
- se o nome do arquivo for diferente, use `--input_csv=...`
- mantenha o arquivo como dado bruto; qualquer derivação deve ir para `data/interim/`, `data/processed/` ou `outputs/`
