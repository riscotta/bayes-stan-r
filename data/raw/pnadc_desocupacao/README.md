# PNAD Contínua — desocupação

Arquivos versionados para permitir reprodução imediata do estudo em `scripts/pnadc_desocupacao/pnadc_desocupacao_rstan.R`.

## Conteúdo

- `pnadc_mensal_taxa_desocupacao_6381.csv`: série mensal total.
- `pnadc_trimestral_taxa_desocupacao_por_sexo_4093.csv`: série trimestral por sexo, incluindo a linha `Total` para checagem de coerência.
- `pnadc_mensal_taxa_desocupacao_jan_2026.csv`: extração pontual de 202601 usada como checagem opcional de consistência.

## Observação

Os arquivos usam separador `;` e cabeçalho compatível com o script do estudo.
