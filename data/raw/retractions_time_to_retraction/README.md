# Retratacoes cientificas — base bruta externa

Este estudo usa a base **Global Scientific Retractions 1927–2026**.

O CSV bruto **nao e versionado neste repositorio**. Para reproduzir a analise, baixe a base em:

`https://www.kaggle.com/datasets/kanchana1990/global-scientific-retractions-19272026`

Salve o arquivo como:

`data/raw/retractions_time_to_retraction/global_scientific_retractions_1927_2026.csv`

Script associado:

- `scripts/retractions_time_to_retraction/retractions_time_to_retraction_cmdstanr.R`
